---

## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | Pest (PHP) + Playwright (E2E smoke, manual-or-scripted) |
| **Config file** | `phpunit.xml` (pest.config under the hood) |
| **Quick run command** | `php artisan test --filter=Role` (scoped to role-related Feature + Unit tests) |
| **Full suite command** | `php artisan test` |
| **Estimated runtime** | ~25s scoped / ~90s full (current suite: 543 tests, ~85s on this machine) |
| **Type-check command** | `./vendor/bin/phpstan analyse --no-progress --memory-limit=512M` (baseline target: 0 net new) |
| **Style command** | `./vendor/bin/pint --test` (must be clean before each PR-eligible commit) |

---

## Sampling Rate

- **After every task commit:** Run `php artisan test --filter=Role` (scoped, ~25s) — catches RBAC regressions immediately.
- **After every plan wave:** Run `php artisan test` (full suite, ~90s) — catches regressions in unrelated tests caused by behavioral changes (CLAUDE.md "three layers of testing" rule).
- **After Wave 0 (foundation):** Also run `./vendor/bin/phpstan analyse` to verify baseline = 0.
- **Before `/gsd:verify-work`:** Full Pest suite green, PHPStan baseline = 0, Pint clean, Playwright smoke complete for all 4 seeded accounts.
- **Max feedback latency:** 90 seconds.
