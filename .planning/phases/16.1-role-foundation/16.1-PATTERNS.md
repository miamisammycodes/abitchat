### `app/Http/Middleware/RequireSuperAdmin.php` (middleware, NEW)

**Analog:** `app/Http/Middleware/AdminAuthenticate.php` (entire file, lines 1-25) — this is what `RequireSuperAdmin` REPLACES.

**Existing middleware shape** (the file being deleted):

```php
<?php

declare(strict_types=1);

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Auth;
use Symfony\Component\HttpFoundation\Response;

class AdminAuthenticate
{
    public function handle(Request $request, Closure $next): Response
    {
        if (! Auth::guard('admin')->check()) {
            return $request->expectsJson()
                ? response()->json(['message' => 'Unauthenticated.'], 401)
                : redirect()->guest(route('admin.login'));
        }

        return $next($request);
    }
}
```

**Apply per RESEARCH Pattern 8:**

```php
class RequireSuperAdmin
{
    public function handle(Request $request, Closure $next): Response
    {
        $user = $request->user();
        if (! $user instanceof User || ! $user->hasRole(Role::SuperAdmin)) {
            abort(403, 'Platform admin required.');
        }
        return $next($request);
    }
}
```

Note: 403 abort (not redirect) — `auth` middleware upstream handles the redirect for unauthenticated users; this middleware only handles "authenticated but not super_admin". `shouldRenderJsonWhen` in `bootstrap/app.php` already does content negotiation (per RESEARCH "Don't Hand-Roll" table).

```php
Schema::table('users', function (Blueprint $table) {
    $table->foreignId('tenant_id')->nullable()->after('id')->constrained()->cascadeOnDelete();
```

```php
Schema::create('user_roles', function (Blueprint $table) {
    $table->id();
    $table->foreignId('user_id')->constrained()->cascadeOnDelete();
    $table->enum('role', ['super_admin', 'owner', 'manager', 'agent']);
    $table->foreignId('tenant_id')->nullable()->constrained()->cascadeOnDelete();
    $table->timestamps();
    $table->index(['user_id', 'tenant_id']);
    $table->index(['tenant_id', 'role']);
    $table->unique(['user_id', 'role', 'tenant_id'], 'user_roles_user_role_tenant_unique');
```
