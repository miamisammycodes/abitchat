Migration up() steps:
1. Backfill messages.content with encrypted values (idempotent via try/catch on decryptString):

```
DB::table('messages')
    ->orderBy('id')
    ->chunk(100, function ($messages) {
        foreach ($messages as $msg) {
            $val = $msg->content;
            if ($val === null) continue;
            try {
                Crypt::decryptString($val);
                // Already encrypted — skip
            } catch (\Illuminate\Contracts\Encryption\DecryptException $e) {
                DB::table('messages')
                    ->where('id', $msg->id)
                    ->update([
                        'content'      => Crypt::encryptString($val),
                        'content_hash' => null,  // Neutralize: MD5(plaintext) is a privacy leak

            }
        }
    });
```
